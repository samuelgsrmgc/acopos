32285: INIT broadcast data point: Incompatible cycle times (POWERLINK, NC manager)

Info: Extended coupling data für ACP10_MC_BROADCAST (0/1: No/Yes)
Data type: USINT

Description:
Under which conditions the initialization of a broadcast of data point is possible, depends on which value for the parameter "Extended coupling data for ACP10_MC_BROADCAST=No/Yes" is defined in the NC configuration for the appropriate network interface.

Extended coupling data for ACP10_MC_BROADCAST=No:
The initialization of a broadcast of data point is only then possible, if the following cycle times are equal:
- Cycle time of the NC manager task class
- POWERLINK cycle time

Extended coupling data for ACP10_MC_BROADCAST=Yes:
The initialization of a broadcast of data point is only then possible, if the ratio of the following cycle times is integer:
- Cycle time of the NC manager task class
- POWERLINK cycle time

If the conditions specified above are not fulfilled, this error is indicated.

Reaction:
Processing of the corresponding NC action is aborted.

Cause/Remedy:
Adjust the cycle times listed in the description above, so that the conditions are fulfilled.
