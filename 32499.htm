32499: PLCopen_CyclicData_TaskClass: Values for real and virtual axis are not equal

Info: Channel number of NC objects
Data type: USINT

Description:
For SG4 targets in the NC mapping table for each ACOPOS communication channel that task class can be defined, in which the cyclic PLCopen data are to be copied from a program to the POWERLINK network and vice versa. This is possible in the data record of a real or virtual axlis in the column "Additional Data" with the following XML attribute:
- PLCopen_CyclicData_TaskClass=""

The real and virtual axis with the same node number and the same channel number are operated using the same ACOPOS communication channel. This error is output if different task classes are defined for the real and the virtual axis from the same communication channel. Moreover, the values defined with PLCopen_CyclicData_TaskClass are entered in the network trace for the corresponding NC objects with ParID TASKCL_CycDat_PLCopen.

Reaction:
The ACOPOS startup function is aborted for this ACOPOS.

Cause/Remedy:
Correct the values defined with PLCopen_CyclicData_TaskClass in the NC mapping table.
