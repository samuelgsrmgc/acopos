32262: Trace test data point: The NC object is invalid

Info: Index of test data point
Data type: UINT

Description:
This NC object is not permitted for use in a Multi Axes Trace. Only NC objects with the following types may be used:
- Real axis
- Virtual axis

Reaction:
Abortion of trace processing.

Cause/Remedy:
Remove the invalid NC object from the trace configuration or replace it by a valid one.
