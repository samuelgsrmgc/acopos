35076: SMC Fail Safe: Homing - reference pulse : Max. trigger speed is too big

Info:
Data type:

Description:
Configuration error of encoder homing. The maximum trigger speed is too large to emulate a reference pulse.

Reaction:
The module changes into Fail Safe state.

Cause/Remedy:
Check the encoder unit system parameters!
Check the homing parameter "Maximal trigger speed"!
Carry out a Power Down/Power Up cycle!
