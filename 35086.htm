35086: SMC Fail Safe: Safe output - Test state has changed

Info: Output state
Data type: UDINT

Description:
Error in the safe outputs.

The output state of STO1, STO2, and SBC is displayed in the additional information (info0, info1, info2).
State is between 0x00 and 0x0F: Module defect.
State is between 0x21 and 0x23: Internal error detected.

Offset information for safety logger:
binary data[17]: Info0 = State of STO1 channel
binary data[18]: Info1 = State of STO2 channel
binary data[19]: Info2 = State of SBC channel

Reaction:
The module changes into Fail Safe state.

Cause/Remedy:
channel specific state:
0x00 to 0x0F: Module defect. Replace the module!
0x21 to 0x23: nternal error detected. Call technical support!
