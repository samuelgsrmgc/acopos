32286: Network coupling: Incompatible cycle times (network interfaces, NC task class)

Info: Extended coupling data für ACP10_MC_BROADCAST (0/1: No/Yes)
Data type: USINT

Description:
Under which conditions the coupling of axes on different networks is possible, depends on which value for the parameter "Extended coupling data for ACP10_MC_BROADCAST=No/Yes" is definied in the NC configuration for the appropriate network interface.

Extended coupling data for ACP10_MC_BROADCAST=No:
The coupling of axes on different networks is only then possible, if the following conditions are fulfilled:
- The network cycle time of the slave axis is equal to the cycle time of the NC manager task class
- The network cycle time of the slave axis is greater than or equal to the network cycle time of the master axis
- The ratio of the network cycle times of slave and master axis is integer

Extended coupling data for ACP10_MC_BROADCAST=Yes:
The coupling of axes on different networks is only then possible, if the ratio of the following cycle times is integer:
- The cycle time of the NC manager task class
- The network cycle time of the slave axis
- The network cycle time of the master axis

If one of the conditions specified above is not fulfilled, this error is indicated.

Reaction:
Processing of the corresponding NC action is aborted.

Cause/Remedy:
Adjust the cycle times listed in the description above, so that all conditions are fulfilled.
