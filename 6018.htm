6018: Hardware: 15V power supply fail

Info:
Data type:

Description:
Runtime error in the hardware monitor using the 'switch on controller' command or while the controller is active. See also error number 4005.
The internal 15V supply voltage is defective. Depending on the hardware this supply is used for the plug-in cards, fans, current measurement, DC bus relay, etc.

Reaction:
See error number 6021.

Cause/Remedy:
Failure or voltage dip in the 24VDC supply voltage for the servo drive.
Failure or voltage dip in the mains supply for the servo drive.
ACOPOS hardware defect.
