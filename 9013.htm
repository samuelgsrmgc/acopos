9013: Temperature sensor (Motor|Choke|External): Short circuit

Info: Temperature
Data type: REAL

Description:
Runtime error when using a linear thermistor if the lower resistance limit of the temperature sensor MIN(MOTOR_TEMPSENS_PAR1, MOTOR_TEMPSENS_PAR2) is not reached.
Runtime error when using a PTC switch if the resistance of the PTC switch falls below 30Ohm.

Reaction:
See error number 9000.

Cause/Remedy:
Motor temperature sensor on connector X4 wiring error or defect: Temperature sensor short circuit. When the controller is switched off, compare the environmental temperature with TEMP_MOTOR. -> Check the resistance of the motor temperature sensor on connector X4 on the X4 terminal for plausibility using a multimeter.
The lower resistance limit of the temperature sensor MIN(MOTOR_TEMPSENS_PAR1, MOTOR_TEMPSENS_PAR2) was selected too high.

Problem with the motor temperature measurement: See error number 9010.
