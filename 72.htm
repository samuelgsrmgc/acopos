72: firmware version less than minimal necessary firmware version

Info: minimal version
Data type: UINT

Description:
The firmware version downloaded to the target is less than the necessary firmware version of the device.

Reaction:
The downloaded firmware will not be burned to the device memory.

Cause/Remedy:
The minimal necessary firmware version of the device is greater than the downloaded firmware version.
The ACP10 version of the project should be greater than the minimal version of the device.
