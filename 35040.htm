35040: SMC Fail Safe: SLS3 - Speed limit is out of range

Info:
Data type:

Description:
Configuration error of the safety function SLS3. The SLS3 speed limit was configured out of range. The internally scaled speed limit has exceeded the limit INT16.

Reaction:
The module changes into Fail Safe state.

Cause/Remedy:
Correct the configuration in SafeDESIGNER and carry out a Power Down/Power Up cycle!
