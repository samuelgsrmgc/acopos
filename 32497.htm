32497: Task class for handling of cyclic data with PLCopen in acp10cfg is invalid

Info: Maximum task class number
Data type: USINT

Description:
For SG4 targets in the ACP10 configuration (acp10cfg) for each POWERLINK interface with the parameter "Task class for handling of cyclic data with PLCopen (only for SG4)" that task class can be selected, in which the cyclic PLCopen data are to be copied from a program to the POWERLINK network and vice versa.

If the BR module acp10cfg containes an invalid value for this parameter, then this error is displayed with the maximum task class number as additional information. Moreover, the value contained in acp10cfg is entered in the network trace for each POWERLINK interface with node number 0 with ParID TASKCL_CycDat_PLCopen.

Reaction:
The ACOPOS startup function is aborted for all ACOPOS modules connected to this POWERLINK interface.

Cause/Remedy:
Correct the value in the ACP10 configuration.
