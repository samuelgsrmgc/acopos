35095: SMC Fail Safe: Encoder - Configuration error

Info: Additional info
Data type: UDINT

Description:
Configuration error of the encoder unit system.

The additional information (Info1) shows the details:
0x00000001 ... The module parameter "Count of physical reference system" is greater than the module parameter "Units per count of physical reference system".
0x00000002 ... The module parameter "Normalized maximal speed" is lower than "Units per count of physical reference system" / ("Count of physical reference system" * 800us * 2^16).
0x00000003 ... The used encoder has a resoultion lower than 10^3 increments per meter.
0x00000004 ... The used encoder has a resoultion higher than 10^12 increments per meter.
0x00000005 ... The configured unit system has a higher resolution than the increment resolution of the encoder.
0x00000006 ... One configured unit is shorter than one nm.

Offset information for safety logger:
binary data[20..17]: Info1


Reaction:
The module changes into Fail Safe state.

Cause/Remedy:
Check the configuration in the SafeDESIGNER!
