35250: SMC: Encoder - EnDat Master encoder error register is set

Info: Fehler register
Data type: UDINT

Description:
Error at the encoder input. In the EnDat master error register an error has been set.

The additional information (Info1) shows the details:
Bit0 (0x00000001) ... Encoder alarm F1
Bit1 (0x00000002) ... Encoder alarm /F2
Bit3 (0x00000008) ... Pos1/Pos2 compare
Bit4 (0x00000010) ... Consecutive number of position 2
Bit5 (0x00000020) ... Watchdog
Bit6 (0x00000040) ... Position1 CRC5
Bit7 (0x00000080) ... Position2 CRC5
Bit10 (0x00000400) ... Mono flop time
Bit11 (0x00000800) ... Pos1/Pos2 compare
Bit12 (0x00001000) ... Consecutive number of position 2
Bit13 (0x00002000) ... Watchdog
Bit14 (0x00004000) ... Position1 CRC5
Bit15 (0x00008000) ... Position2 CRC5
Bit24 (0x01000000) ... Error type 1
Bit25 (0x02000000) ... Error type 2
Bit26 (0x04000000) ... Error type 3
Bit27-29 ... Number of Pos1/Pos2 compare errors
Bit30 (0x40000000) ... EnDat Strobe error
Bit31 (0x80000000) ... EnDat transfer completed

Offset information for safety logger:
binary data[20..17]: Info1

Reaction:
The status SafePositionValid is set to FALSE.
The status NotErrENC is set to FALSE.
If a safety function is used which needs an encoder, then the module changes into Functional Fail Safe state.

Cause/Remedy:
The data transfer is disturbed between the encoder and the module. Check the wiring between the encoder and module!
The encoder is defect. Change the encoder!
The module is defect. Change the module!
Call technical support!
