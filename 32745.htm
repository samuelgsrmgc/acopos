32745: SDM_Motion_Action not allowed (Motion Trace already started by application)

Info:
Data type:

Description:
For processing of this SDM_Motion_Action the Motion Trace must not be active.

Reaction:
Processing of this SDM_Motion_Action is aborted.

Cause/Remedy:
The Motion Trace was already started by the application.
Call the SDM Funktion for starting the Motion Trace later again.
Change of the Automation Runtime version (starting the Motion Trace during it is active should be blocked).
