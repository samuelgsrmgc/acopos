64007: Timeout for enable of acyclic network communication

Info: Timeout [sec]
Data type: UDINT

Description:
During the network initialization (function for ACOPOS startup) for the POWERLINK network, before reading the parameter BOOT_STATE for each ACOPOS module is waited, until one of the the following states is indicated by the plState() function:
- plSTATE_STATION_ACTIVATEING (0x0004: The station has just been activated)
- plSTATE_STATION_ACTIVE (0x0005: Station is online and cyclic copy instructions are carried out)
- plSTATE_STATION_CHANGED (0x0007: The hardware configuration for the station was changed)
If the timeout for waiting on this POWERLINK state is reached, then this warning appears.

Reaction:
This warning is indicated.

Cause/Remedy:
Overload, malfunction or interruption of the network connection or ACOPOS failure.
See also error number 1004.
See also error number 1012.
