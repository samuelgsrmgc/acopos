35240: SMC Functional Fail Safe: Violation of safety function

Info: Additional info
Data type: UDINT

Description:
Error during an active safety function.

The additional information (Info1) shows the details:
0x00000000 ... Statemachine is in State Functional Fail Safe
0x00000010 ... Violation of standstill position during active SS2 or SOS
0x00000020 ... Encoder error was detected (encoder is mandatory)
0x00000030 ... Violations of the direction, both directions (positive and negative) are monitored.
0x00000031 ... Violation of the positive direction during active SDI.
0x00000032 ... Violation of the negative direction during active SDI.
0x00000041 ... Violation of standstill speed limit during SLI request.
0x00000042 ... Violation of Increments during active SLI
0xSSSS0050 ... Violation of the actual speed limit, no active speedlimit! (SSSS.. Speed at violation)
0xSSSS0051 ... Violation of the actual speed limit, ramp monitoring is active (SSSS.. Speed at violation)
0xSSSS0052 ... Violation of the actual speed limit, SMS is active (SSSS.. Speed at violation)
0xSSSS0053 ... Violation of the actual speed limit, SOS is active (SSSS.. Speed at violation)
0xSSSS0054 ... Violation of the actual speed limit, SS2 is active (SSSS.. Speed at violation)
0xSSSS0055 ... Violation of the actual speed limit, SLS1 is active (SSSS.. Speed at violation)
0xSSSS0056 ... Violation of the actual speed limit, SLS2 is active (SSSS.. Speed at violation)
0xSSSS0057 ... Violation of the actual speed limit, SLS3 is active (SSSS.. Speed at violation)
0xSSSS0058 ... Violation of the actual speed limit, SLS4 is active (SSSS.. Speed at violation)
0x00000059 ... Alive test of set position for requested safety functions missing
0x00000060 ... Warning on output was not acknowledged
0x00000061 ... Timeout for alive test of set position
0x00000062 ... Homing timeout for safety function SMP exceeded
0x00000063 ... Violation of the deceleration ramp for SMP
0x00000064 ... Violation of the deceleration ramp for SLP
0x00000065 ... Violation of the SMP position window in negative direction
0x00000066 ... Violation of the SMP position window in positive direction
0x00000067 ... Violation of the SLP position window in negative direction
0x00000068 ... Violation of the SLP position window in positive direction
0x00000069 ... Movement in neg. direction and lower SMP limit violated
0x0000006A ... Movement in pos. direction and upper SMP limit violated
0x0000006B ... SLP requested but axis not homed
0x0000006C ... general error regarding SMP / SLP
0x000000F7 ... Encoder absolute position exceeded INT32 range

Offset information for safety logger:
binary data[20..17]: Info1

Reaction:
The module changes into Functional Fail Safe state.

Cause/Remedy:
Check the application and/or the encoder!
