64009: Acp10cfg contains a POWERLINK interface, for which no axis is defined

Info: Index of POWERLINK interface in Acp10cfg (name in Logger)
Data type: UINT

Description:
If a POWERLINK interface is contained in Acp10cfg, for which no drive node is configured via hardware tree or NC mapping table, then this warning is displayed. Additionally the following informations are registered in the Logger:
- POWERLINK interface in Acp10cfg
- Interface name
- No drive node defined

Reaction:
This warning is indicated.

Cause/Remedy:
Delete this POWERLINK Interface from Acp10cfg.
