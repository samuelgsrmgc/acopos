32271: ACOPOS coupling: The channel number of the receive data is invalid

Info: Channel number of the receive data
Data type: USINT

Description:
Only values 1 to 5 are valid for the channel number of the receive data. This error is reported if a different channel number is specified.

Note:
The INFO is displayed from V2.370 on.

Reaction:
Processing of the corresponding NC action is aborted.

Cause/Remedy:
Specify valid channel number for configuration of the receive data.
