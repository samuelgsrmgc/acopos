31260: Current axis configuration only possible in simulation mode

Info:
Data type:

Description:
Response error using the 'switch on controller' command.
The SDC axis being used can only be used in simulation mode. The SDC axis is either just a simulated one or no SDC axis configuration PV was provided.

Reaction:
The startup procedure is canceled and the controller is not switched on.

Cause/Remedy:
Check the name of the SDC axis configuration PV, in the event one is present and the SDC axis is supposed to be more than just a simulated axis.
Check the entry in the 'Additional Data' field in the NC-Mapping table if the SDC axis is supposed to be more than just a simulated axis.
Enable simulation mode.
