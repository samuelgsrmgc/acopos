9078: Power stage: Temperature sensor 1: Overtemperature - Movement stopped

Info: Temperature
Data type: REAL

Description:
Runtime error if the temperature sensor 1 (TEMP_AIRIN) exceeds the limit (see INFO).

Reaction:
See error number 9000

Cause/Remedy:
The module is operated outside the specifications:
- Continuous current is too large
- Ambient temperature is too high.
- Temperature sensor is defective.

8B: Fan is defective.
8B: The air flow is choked in the area of the inlet air or exit air.
