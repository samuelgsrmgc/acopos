64010: NC software test version (validity period in Logger)

Info:
Data type:

Description:
For NC software test versions this warning is entered in each case in the network trace. In addition appropriate information and with "Valid from" the beginning and with "Valid to" the end of the validity period is registered in the Logger as follows:
- NC software test version
- Valid from YYYY-MM-DD HH:MM:SS
- Valid to YYYY-MM-DD HH:MM:SS

Reaction:
During the startup the NC manager compares the date/time value of the PLC with the values of the validity period. If the value of the PLC is not below the value displayed with "Valid from" and not over the value displayed with "Valid from", then the test version functions exactly the same as a standard version (apart from the warnings mentioned above). Otherwise this warning is displayed in the NC structure of each NC object.

Cause/Remedy:
Use the corresponding NC software standard version instead of the test version.
