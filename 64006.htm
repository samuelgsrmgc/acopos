64006: Drive did not become synchronous with network master

Info: Timeout [sec]
Data type: UDINT

Description:
At the end of the network initialization (function for ACOPOS startup) for the POWERLINK network, after start of the operating system for each ACOPOS module is waited, until the following bit in the cyclic status (parameter STATUS_CYCLIC_BITS) is set:
- Bit20: Drive synchronous
If the timeout for waiting on this status is reached, then this warning appears.

Reaction:
This warning is indicated.

Cause/Remedy:
Overload, malfunction or interruption of the network connection or ACOPOS failure.
See also error number 1004.
See also error number 1012.
