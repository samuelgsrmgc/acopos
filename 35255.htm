35255: SMC Fail Safe: Encoder - Acceleration limit exceeded

Info:
Data type:

Description:
Error at the encoder input. Encoder acceleration limit exceeded!

Reaction:
The status SafePositionValid is set to FALSE.
The status NotErrENC is set to FALSE.
If a safety function is used which needs an encoder, then the module changes into Functional Fail Safe state.

Cause/Remedy:
The connection between the motor and encoder can slip through.
Check the parameter "Maximum acceleration"!
Reduce the encoder acceleration!
Check the mechanical connection between the encoder and the motor!
