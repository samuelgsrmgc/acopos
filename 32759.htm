32759: SDM_Motion_Action: The NC object ident must be zero for this action

Info:
Data type:

Description:
This SDM_Motion_Action is only implemented for NC object ident zero.

Reaction:
Processing of this SDM_Motion_Action is aborted.

Cause/Remedy:
The used Automation Runtime version calls this SDM_Motion_Action with a wrong NC object ident.
Change of the Automation Runtime version.
