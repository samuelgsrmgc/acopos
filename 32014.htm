32014: Operating system version on the drive not compatible with NC manager version

Info: Operating system version on the drive
Data type: UINT

Description:
This error is displayed during the ACOPOS startup function if the NC manager detects that the version of the operating system on the ACOPOS is not compatible to the version of the NC manager.

Note:
The version of the operating system on the drive can be determined by viewing the decimal value, which was output as UINT, as a hexadecimal value. For example, the value "0x1234" would correspond with version "1.23.4".

Reaction:
The ACOPOS startup function is aborted for this ACOPOS.

Cause/Remedy:
"acp10sys" does not contain an operating system for the device type of the connected ACOPOS hardware module.
The first three places in the version number of "acp10sys" and "acp10man" are not the same.
See also error number 32013.
