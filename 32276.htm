32276: Network coupling: Cycle times not equal (network interfaces, NC task class)

Info:
Data type:

Description:
If coupling master and coupling slave are connected to different network interfaces, it is proofed, wether the following cycle times are equal:
- Cycle time of the NC manager task class
- POWERLINK cycle time (if master or slave are operated via POWERLINK network)

If this is not the case, this error is indicated.

Reaction:
Processing of the corresponding NC action is aborted.

Cause/Remedy:
Equate the cycle times, which are listed in the description above.
