31248: Trigger Interface: HW Module not OK

Info: TrigIf Index
Data type: UINT

Description:
Runtime error in the trigger hardware monitoring using the 'switch on controller' command or while the controller is active.
The SDC PV 'iLifeCnt' of the trigger interface is not changed with each SDC cycle. The trigger hardware module does not provide new values with each SDC cycle.

Reaction:
See error number 31249.

Cause/Remedy:
Check the SDC trigger interface specific PVs.
See the procedure for error number 31249.
