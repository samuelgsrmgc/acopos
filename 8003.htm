8003: Table index not valid

Info:
Data type:

Description:
Response error while writing the SEP_ULONG_WR and SEP_FLOAT_WR parameters or while reading the SEP_ULONG_RD and SEP_FLOAT_RD parameters.

Reaction:
None.

Cause/Remedy:
The access counter is incremented after each read and write procedure for the above parameters.
Caution: Internal parameter for writing the EEPROM. False usage can cause damage to the ACOPOS servo drive.
