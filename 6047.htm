6047: Holding brake: Control signal off and output status on

Info:
Data type:

Description:
Runtime error during holding brake control monitor is active. The holding brake was supplied 500ms with current and voltage while the holding brake gating signal was off.

Reaction:
See error number 6029.
But if the position control is active (CONTROLLER_MODE=1 or 3): Movement stopped with the axis limits and switching off the controller after the error removed.

Cause/Remedy:
ACOPOS defective (Holding brake control, signal level monitoring, internal relay)
