39024: EnDat encoder: Operating status error sources: M ALL Power down

Info: Slot
Data type: USINT

Description:
Runtime error in an encoder with activated monitor (activation, see error number 7030). Battery charge too low.
The battery voltage dropped below a certain limit while using a battery-buffered encoder and while the drive was switched-off drive.
A change to the multi-turn position will not be recorded if the encoder is not supplied with power. Therefore, the multi-turn position is invalid.

Reaction:
See error number 7030.

Cause/Remedy:
Change encoder battery.
No encoder supply or supply contains an error.
See wiring guidelines error number 7030.
The error can only be acknowledged by setting PARID_ENCOD_CMD = 1.
The multi-turn position must then be measured.
