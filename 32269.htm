32269: Trace: Too many test data points defined for one trace channel

Info: Max. number of test data points
Data type: UINT

Description:
For one trace channel max. 10 test data points may be defined.
To one trace channel belong the real and the virtual axis with the same channel number on one ACOPOS module.

Reaction:
Abortion of trace processing.

Cause/Remedy:
Reduction of test data points.
