7402: Processing of parameter sequence aborted: Write error

Info: Index of parameter
Data type: UINT

Description:
Runtime error during initialization of a parameter sequence if the parameter sends an error at the present index (index begins at 0).

Reaction:
The subsequent parameters in the parameter sequence are not processed. The status remains at 'not initialized'. The parameter error is additionally entered as runtime error.

Cause/Remedy:
According to the parameter ID and the additionally entered error number.
