35096: SMC Fail Safe: Homing - Configuration error

Info: Additional info
Data type: UDINT

Description:
Configuration error of the homing unit.

The additional information (Info1) shows the details:
0x00000001 ... Maximal trigger speed is out of range.
0x00000002 ... Homing mode "Reference Switch" selected, but no reference switch connected.
0x00000003 ... Homing mode "Home Offset" or "Home Offset with Correction" selected, but no absolute encoder connected.
0x00000004 ... Homing mode "Home Offset with Correction" selected, but SMP not configured.
0x00000006 ... SMP is configured and SMP range is greater than max. safe abs. encoder range.
0x00000007 ... SLP is configured and SLP range is greater than max. safe abs. encoder range.
0x00000008 ... Plugged encoder doesn't support a safe reference pulse.
0x00000009 ... Speed tolerance is out of range.
0x0000000A ... Max. trigger speed is too big to emulate reference pulses.
0x0000000B ... Homing bit of control word is set but not used.
0x0000000C ... RefSwitch bit of control word is set but not used.
0x0000000D ... Homing position is out of SMP position window.
0x0000000E ... Homing mode "Direct with reference pulses" selected, but not supported.

Offset information for safety logger:
binary data[20..17]: Info1

Reaction:
The module changes into Fail Safe state.

Cause/Remedy:
Check the configuration in the SafeDESIGNER!
Check the safety application!
