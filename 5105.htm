5105: Master trigger FIFO full

Info:
Data type:

Description:
Runtime error when cam automat is active (CMD_CAM_START) if the trigger FIFO for the master axis is full.

Reaction:
See error number 4008.

Cause/Remedy:
Too many false triggers when a FIFO length is at the maximum limit (CAM_MA_S_TRIG large and (CAM_MA_S_SYNC + CAM_MA_S_COMP) small). This mostly happens if the trigger window is larger than the half interval (CAM_MA_TRIG_WINDOW> (CAM_MA_S_SYNC + CAM_MA_S_COMP) / 2).
