32193: For this NC object type no INIT parameter module is present

Info:
Data type:

Description:
In order to process the NC action "ncGLOBAL,ncSAVE" or "ncSETUP+ncCONTROLLER,ncSAVE" for an NC object type, at least one INIT parameter module must be present for this NC object type on the target system during startup, which is assigned to an NC object in an NC mapping table. Otherwise, this error is displayed when calling the NC action "ncGLOBAL,ncSAVE" or "ncSETUP+ncCONTROLLER,ncSAVE".

Reaction:
Processing of NC action "ncGLOBAL,ncSAVE" is aborted.

Cause/Remedy:
An INIT parameter module for this NC object type must be assigned to an NC object in an NC mapping table and transferred to the target system.
