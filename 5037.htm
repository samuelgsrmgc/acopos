5037: Homing procedure mode not possible: Wrong encoder type

Info:
Data type:

Description:
Response error using 'start homing procedure' command. A homing mode was defined (ncRESTORE_POS), which requires an absolute encoder measurement system - at least within one revolution.

Reaction:
The homing procedure is not started.
The 'homing position valid' state is reset.

Cause/Remedy:
Incorrect homing mode.
Incorrect encoder type.
-> Use absolute encoder or at least single turn encoder instead of incremental encoder.
