35030: SMC Fail Safe: Encoder mismatch - Configuration data changed several times

Info:
Data type:

Description:
Error during startup of the encoder in the module state PreOperational. A change in the encoder configuration data has been detected. The encoder configuration data has been changed several times.

Reaction:
The module changes into Fail Safe state.

Cause/Remedy:
Under electrical voltage, a exchange of an encoder is not allowed.
Carry out a Power Down/Power Up cycle!
