32001: Error calling CAN_xopen()

Info: Status of CAN_xopen()
Data type: UINT

Description:
For the operation of ACOPOS servo drives via the CAN network, the function CAN_xopen() is called for each CAN interface defined in the NC configuration to initialize the required resources of the CAN driver. This error number is output if an error occurs with this procedure.

Reaction:
Initialization of all ACOPOS nodes connected to this CAN interface is aborted. All corresponding NC objects can no longer be operated.

Cause/Remedy:
Wrong interface name in the NC configuration.
Status of CAN_xopen(): See CAN library error numbers.
