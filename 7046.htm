7046: Resolver: Cable disturbance

Info: Slot
Data type: USINT

Description:
Runtime error in a resolver with activated monitor (activation, see error number 7030). A vector comparison is made between the sine/cosine signals in the resolver evaluation circuit. An error is output if deviations are too large. Correct functioning of the monitor depends greatly on the input impedance and the voltage ratio of the resolver.

Reaction:
See error number 7030.

Cause/Remedy:
See error number 7045.
If the resolver does not meet the specifications of the AC122 plug-in card, the monitor of the cable disturbances can be deactivated by setting bit 0 of ENCODX_LINE_CHK_IGNORE.
Caution: This can cause uncontrolled movements when 'real' resolver errors occur.
This error is not supported by 8AC122.60-1 plug-in cards.
