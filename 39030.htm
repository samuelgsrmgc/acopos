39030: Encoder: Warning bit is set

Info: Slot
Data type: USINT

Description:
Runtime info for an SSI or BiSS encoder with evaluation of warning bits enabled.

Reaction:
None

Cause/Remedy:
An event occurred that caused the encoder to set the warning bit.
The cause of the warning bit can be found in the documentation provided by the encoder manufacturer.
Encoder is configured incorrectly.
Encoder connection is interrupted or defective. See wiring guidelines error number 7030.
