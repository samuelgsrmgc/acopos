6061: CTRL Speed controller: Limit speed exceeded

Info: Limit speed
Data type: REAL

Description:
Runtime error when mechanical shaft speed (SCTRL_SPEED_ACT) exceeds 110% of the maximum permitted speed.
The maximum permitted speed is added to the additional error information.

The maximum permitted speed is determined by the motor parameter (MOTOR_SPEED_MAX). This value can be reduced by using field weakining (PMSM).
See Parameter initialization/calculation


Reaction:
See error number 9000.

Cause/Remedy:
Incorrect parameter setting
Avoid high speeds
