32253: The task class for POWERLINK output cycle trigger is invalid

Info: Maximum task class number
Data type: USINT

Description:
In the POWERLINK configuration a task class can be selected as output cycle trigger (by default the output cycle trigger is carried out by the system tick). If for a POWERLINK interface a higher task class than the task class of the NC manager task is selected, then this error is indicated.

Reaction:
The ACOPOS startup function is aborted for all ACOPOS modules connected to this POWERLINK interface.

Cause/Remedy:
Select a valid task class in the POWERLINK configuration.
