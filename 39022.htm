39022: EnDat encoder: Warning bit is set

Info: Slot
Data type: USINT

Description:
Runtime error for an EnDat 2.2 encoder.
The encoder sets the warning bit, which is a summation bit for multiple causes.

Reaction:
keine

Cause/Remedy:
The warning bit generally outputs multiple error messages that describe the cause:
Possible other error messages:
39011, 39012, 39013, 39014 and 39015.
Once the cause of the errors has been corrected, you can try clearing the warning bit with ENCOD_CMD = 6.
Important: This command also clears any set alarm bits!
