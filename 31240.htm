31240: Homing procedure mode not allowed with current HW Type

Info:
Data type:

Description:
Response error using 'start homing procedure' command. A homing mode was configured that is not supported for SDC axes ('ncDCM' or 'ncDCM + ncCORRECTION').

Reaction:
The homing procedure is not started.
The 'homing position valid' state is not set.

Cause/Remedy:
Use a homing mode other than 'ncDCM' or 'ncDCM + ncCORRECTION'.
