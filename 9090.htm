9090: Motor encoder temperature sensor: Temperature value not valid

Info:
Data type:

Description:
Runtime error if the temperature sensor in the motor encoder (ENCODxTEMP) returns an invalid value.

Reaction:
Siehe Fehler-Nummer 9000

Cause/Remedy:
- Data transmission from the encoder to the drive is not active yet.
- The motor encoder and/or the plug-in module do not support the temperature measurement in the encoder and the temperature monitoring is falsely activated (MOTOR_ENCOD_TEMP_LIM! = 0).
- False triggering due to defective hardware or malfunctions (broken connection, line termination, wiring error, control hardware defective or failed, ACOPOS 24VDC supply voltage, malfunctions). See also wiring guidelines error number 7030.
- The temperature sensor in the encoder is defective.
