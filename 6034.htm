6034: Cyclic set value mode aborted: Set speeds missing

Info:
Data type:

Description:
Runtime error while speed control is active with cyclic set values. No speed set value was received from the network for at least 3 master cycles.

Reaction:
See error number 4007.

Cause/Remedy:
Drive is not supplied with enough speed set values -> check application program.
See error number 5110.
