31249: Drive Interface: HW Module not OK

Info:
Data type:

Description:
Runtime error in the drive hardware monitoring using the 'switch on controller' command or while the controller is active.
The SDC PV 'iLifeCnt' of the drive interface is not changed with each SDC cycle. The drive hardware module does not provide new values with each SDC cycle.

Reaction:
When 'switch on controller' command:
The startup procedure is canceled and the controller is not switched on.

When controller is active:
Movement stopped with the axis limits and by switching off the controller.
Setting STOP_ERR_DECEL_RAMP=ncCTRL_OFF: Movement is aborted by switching off the controller immediately. oDriveEnable is used to request the power section shutoff.

Cause/Remedy:
Check the SDC drive interface specific PVs.

Procedure:
1. Check SDC axis configuration PV
2. Check the I/O mapping for the PVs.
3. Check the error states of the hardware module on which the SDC PVs are mapped.
4. Check the time settings.

Maloperation due to defective hardware or interference (broken connection, line termination, wiring error, control hardware defective or failed).
