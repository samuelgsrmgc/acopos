5301: Start cam automat linkage not possible: Parameter outside the valid range

Info: Parameter ID
Data type: 'PARID'

Description:
Response error at start of the cam automat (CMD_AUT_START) if an invalid configuration was detected.

Reaction:
The cam automat start is not executed.

Cause/Remedy:
The parameter ID of the additional information differentiates between the following causes:
AUT_COMP_MODE
Invalid compensation gears mode in the start state.
