1027: Function is not available for this hardware

Info:
Data type:

Description:
Response error while writing a parameter, if the current ACOPOS hardware does not support the requested functinality.

Reaction:
The parameter value is not accepted and the requested function is not executed.

Cause/Remedy:
Wrong ACOPOS hardware being used for the desired function.
See ACP10 Software NC Object ncAXIS: ACOPOS Hardware Information
See also error number 1002.
