7222: Power stage: Summation current connection X5: Overcurrent (Ground fault)

Info: Limit
Data type: REAL

Description:
Runtime error if the absolute value of the summation current (= |VCTRL_I0| = |VCTRL_I1 + VCTRL_I2 + VCTRL_I3|) exceeds the module-dependent limit (see info):

Reaction:
See error number 7224.

Cause/Remedy:
8BVPxxxxxxxx.xxx-x: Ground is connected to the X5 connection on contact L1, L2 or L3
-> Check voltage using multimeter.

8BVPxxxxxxxx.xxx-x: An inverter module with ground fault on the motor side is connected via the DC bus.
-> Check whether the 8BVIxxxxxxxx.xxx-x:X5 connection is connected to the ground.

8BVPxxxxxxxx.xxx-x: Main filter wiring error: Connection X1 and X2 switched
-> Check whether the 8BVFxxxxxxxx.xxx-x:X1 connection is connected to the power mains.
-> Check whether the 8BVFxxxxxxxx.xxx-x:X2 connection is connected to the 8BVRxxxxxxxx.xxx-x:U1,V1,W1 connection.

8BVPxxxxxxxx.xxx-x: Main filter defect: Replace main filter

8BVIxxxxxxxx.xxx-x: Ground is connected to the X5 connection on contact U, V or W

8BVIxxxxxxxx.xxx-x: Capacitive load: When using long motor cable (> 50m) or cable form third-party manufacturer.
