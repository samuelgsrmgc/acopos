6057: Position loop controller: Load encoder error

Info:
Data type:

Description:
Runtime error in an encoder with activated monitor (activation, see error number 7030).

Reaction:
See error number 7030.

Cause/Remedy:
If the encoder status is generated on this axis, then detailed information is provided in an additional entry about an encoder runtime error
If the encoder status is not generated on this axis (encoder on another axis or X20 module), then the corresponding module must provide detailed information.
