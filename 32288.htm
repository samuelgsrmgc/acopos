32288: Extended coupling data not possible (task class tolerance is not zero)

Info: Number of NC Manager task class
Data type: USINT

Description:
Extended coupling data are used then for a network interface, if "Extended coupling data for ACP10_MC_BROADCAST=Yes" is defined for this network interface in the NC configuration.
Extended coupling data can only be used if it is guaranteed that the coupling data can be received by the ACOPOS in each coupling cycle in synchronization with the network. Among other things (see also error number 32287), this requires that the task class tolerance of the NC manager task class is set to zero.

If the task class tolerance of the NC manager task class is not set to zero, then this error is indicated.

Reaction:
Initialization of extended coupling data is aborted.

Cause/Remedy:
Set the task class tolerance for the NC manager task class to zero.
