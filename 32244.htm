32244: No PDO defined in the cyclic frame for this channel: NC object is disabled

Info:
Data type:

Description:
This error is displayed, if no data point is defined for one channel (one axis) of an ACOPOS modul in the Process Data Mapping of the POWERLINK configuration.

Reaction:
The processing of this channel (this axis) is disabled.

Cause/Remedy:
Channel 2 of an ACOPOSmulti module is disabled, because Single Axis is selected as operating mode (this is the default setting for Single Axis ACOPOSmulti with SafeMC). Channel 2 can only be used, if Double Axis is selected as operating mode (or User Configured with activation of data points of axis 2).
