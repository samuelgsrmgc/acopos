9012: Temperature sensor (Motor|Choke|External): Not connected or destroyed

Info: Temperature
Data type: REAL

Description:
Runtime error when using a linear thermistor if the upper resistance limit of the temperature sensor MAX(MOTOR_TEMPSENS_PAR1, MOTOR_TEMPSENS_PAR2) is exceeded.
Runtime error when using a PTC switch if the resistance of the PTC switch exceeds 1MOhm.

Reaction:
See error number 9000.

Cause/Remedy:
Motor temperature sensor on connector X4 wiring error or defect: Motor temperature sensor on connector X4 is not connected, or there is a loose contact. -> When the controller is switched off, compare the environmental temperature with TEMP_MOTOR. -> Check the resistance of the motor temperature sensor on connector X4 on the X4 terminal for plausibility using a multimeter.
The upper resistance limit of the linear thermistor MAX(MOTOR_TEMPSENS_PAR1, MOTOR_TEMPSENS_PAR2) was selected too low.

Problem with the motor temperature measurement: See error number 9010.
