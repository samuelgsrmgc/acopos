6026: Holding brake: Stator current limit exceeded during release

Info:
Data type:

Description:
Runtime error using the 'switch on controller' command when holding brake monitor is active. The maximum current flows while opening (releasing) the holding brake MIN(ACOPOS_CURR_MAX, MOTOR_CURR_MAX).

Reaction:
See error number 6045.

Cause/Remedy:
The current controller is unstable.
The time for opening (releasing) the holding break is too long and the controller is set too 'soft'.
