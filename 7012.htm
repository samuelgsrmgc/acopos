7012: Encoder: Hiperface error bit

Info: Slot
Data type: USINT

Description:
Runtime error in a Hiperface encoder while reading or writing a parameter. An error occurred when transferring data between Hiperface encoder and AC121 plug-in card. As a result, the error bit was set. This error bit is set due to commands which cannot be processed (protocol error, command arguments) or due to internal encoder errors.

Reaction:
Error number 7013 is entered.

Cause/Remedy:
See additionally entered error numbers.
