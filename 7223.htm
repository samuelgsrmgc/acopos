7223: DC bus: Overvoltage DC-GND

Info:
Data type:

Description:
Runtime error if the voltage difference between UDC+ and GND or GND and UDC- is greater than 900V.

Reaction:
See error number 7224.

Cause/Remedy:
Wiring error: When ground fault on motor phases: Check wiring
Invalid mains: IT mains and TN-S mains may not be used
Mains interference: Voltage pulses (e.g.: lightning, etc): Check mains voltage.
8BVxxxxxxxxx.xxx-x: Hardware defect (e.g.: bypass capacitor failure): Replace power module
