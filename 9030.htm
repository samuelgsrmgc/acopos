9030: Junction temperature model: Overtemperature - Movement stopped

Info: Temperature limit
Data type: REAL

Description:
Runtime error if the temperature of the junction temperature model TEMP_JUNCTION exceeds the module-dependent limit TEMP_JUNCTION_LIM (see INFO).


Reaction:
See error number 9000. The junction temperature limiter is switched on during the movement stop.

Cause/Remedy:
Increased current:
- Set value generation incorrect: Speed and acceleration are too high. See error number 5036.
- Load torque too high: Increased current demand due to impermissibly high load torque (friction, jammed mechanics, aging, load thrusts).
- Set current value step: See error number 9300.
- Controller configuration error: Unstable controller.
- Motor configuration or wiring error: MOTOR_COMMUT_OFFSET. -> Check the commutation offset MOTOR_COMMUT_OFFSET.
- Motor defect: Demagnetized motor. -> Check the torque constant MOTOR_TORQ_CONST and the voltage constant MOTOR_VOLT_CONST of the motor.
- Motor dimensioning error: -> Check the current demand for the generation of the required torque. Generally, the torque constant MOTOR_TORQ_CONST is reduced with high currents (> MOTOR_CURR_RATED) due to magnetic saturation.
- Asynchronous motor configuration error: -> Check if the magnetizing current SQRT(2)*MOTOR_MAGNETIZING_CURR is larger than ACOPOS_CURR_RATED.
- Asynchronous motor configuration error: -> Check if the magnetizing current MOTOR_MAGNETIZING_CURR is smaller than 0.5*MOTOR_CURR_RATED.
- 8B0P: Increased charge current by a transient mains failure.

Motor wiring error: See error number 6044.
Motor wiring error: See error number 6045.
Encoder position or speed contains an error: See error number 4014.
Mains disturbance or dimensioning error: Supply voltage or DC bus voltage is too high. Observe the derating which is dependant on the DC bus voltage (see user's manual).
ACOPOS servo drive defect: DC bus voltage measurement problem. See error number 9040.

Measures for minimizing junction heating:
- Switching frequency: ACOPOSmulti: If the switching frequency F_SWITCH is reduced by a factor of 2 and Bit0 is set in the parameter ICTRL_MODE, then the current control performance is maintained with less heating.
- DC bus voltage: ACOPOSmulti - Power supply unit: Reduction of UCTRL_UDC_REF = 750.0 and UCTRL_TN = [0.01 - 0.02]
- DC bus voltage: ACOPOSmulti inverter module: Reduction of UDC_NOMINAL = 750.0
- Current: - Induction motors: KSCTRL_TORQUE_LIM can be used to reduce the current for DC braking.
- Current: Reduce acceleration. Increase jolt time. Reduce motor peak current and motor peak torque.
- Cooling: Climate device used for improved cooling.
- Cooling: ACOPOSmulti: Feed-through cooling: Heat conductive paste used.
