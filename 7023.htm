7023: Encoder: Parameter transfer is active

Info: Slot
Data type: USINT

Description:
Response error while reading/writing encoder OEM data. A read/write access to the encoder OEM data was already activated and is not yet complete.

Reaction:
The command is not executed.

Cause/Remedy:
The interval between two read/write accesses was too short.
Serial communication error (see additionally entered error numbers).
