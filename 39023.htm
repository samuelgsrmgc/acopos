39023: EnDat encoder: CRC error while reading EnDat2.2 additional information

Info: Slot
Data type: USINT

Description:
Runtime error while operating the serial interface of an EnDat2.2 encoder. A data transfer error occurred while sending EnDat2.2 additional information.

Reaction:
The received data is ignored and the data transfer is repeated.

Cause/Remedy:
See also error number 7014.
