35048: SMC Fail Safe: Violation of parameter limits

Info: Additional info
Data type: UDINT

Description:
The parameter limits of at least one parameter in the SafeDESIGNER has been violated.

The additional information (Info1) shows the first parameter where a violation occurred:
0x00000001 ... Count of physical reference system
0x00000002 ... Units per count of physical reference system
0x00000003 ... Counting direction
0x00000004 ... Maximum Speed to normalize the speed range
0x00000005 ... Deceleration Ramp (Units/s^2)
0x00000006 ... Maximum Speedlimit for SMS (Units/s)
0x00000007 ... Safe Speedlimit 1 for SLS (Units/s)
0x00000008 ... Safe Speedlimit 2 for SLS (Units/s)
0x00000009 ... Safe Speedlimit 3 for SLS (Units/s)
0x0000000A ... Safe Speedlimit 4 for SLS (Units/s)
0x0000000B ... Standstill Speed Tolerance (Units/s)
0x0000000C ... Standstill Position Tolerance (Units)
0x0000000D ... Safe Increments (Units)
0x0000000E ... Ramp Monitoring Time for SS1 (us)
0x0000000F ... Ramp Monitoring Time for SS2 (us)
0x00000010 ... Ramp Monitoring Time for SLS1 (us)
0x00000011 ... Ramp Monitoring Time for SLS2 (us)
0x00000012 ... Ramp Monitoring Time for SLS3 (us)
0x00000013 ... Ramp Monitoring Time for SLS4 (us)
0x00000014 ... Delay time to active STO in functional fail safe (us)
0x00000015 ... Delay time to start ramp monitoring (us)
0x00000016 ... Delay time to start SDI (us)
0x00000017 ... Delay time to start SLI (us)
0x00000018 ... Delay time to start SBC (us)
0x00000019 ... Early Limit Monitoring time (us)
0x0000001A ... Encoder monitoring mode
0x0000001C ... Encoder monitoring Speed tolerance (Units/s)
0x0000001D ... Encoder monitoring Position tolerance (Units)
0x0000001E ... Delay time until the brake engages
0x0000001F ... Length of physical reference system for linear encoder (nm)
0x00000020 ... Max. trigger speed (Units/s)
0x00000021 ... Homing monitoring time (us)
0x00000022 ... Homing mode
0x00000023 ... Distance for blocking the activation of "triggering reference pulse"
0x00000024 ... Delay time for start SLP (us)
0x00000025 ... Maximum Acceleration (rad/s^2 or mm/s^2)
0x00000026 ... Homing settings

Offset information for safety logger:
binary data[20..17]: Info1

Reaction:
The module changes into Fail Safe state.

Cause/Remedy:
Check the Safety application!
Check if you are using the correct hwc-file!
