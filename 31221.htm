31221: Encoder error: Cable disturbance or signal disturbance

Info: EncIf Index
Data type: UINT

Description:
Runtime error in the encoder monitoring using the 'switch on controller', 'start homing procedure' command or while the controller is active.
The SDC PV 'iActTime' does not contain a valid time stamp. The time stamp is outside of the allowed time window (no cyclic sampling of actual position).
SDC encoder status PV 'iEncOK' is no longer TRUE.

Reaction:
See subsequent error.
Operation can resume once the SDC PVs 'iActTime' and 'iEncOK' for the encoder interface receive valid data in every SDC cycle.

Cause/Remedy:
Check the I/O mapping for the encoder PVs.
Check the error states of the encoder hardware module on which the SDC encoder PVs are mapped.
Check SDC axis configuration PV.
Encoder is not connected.
No encoder supply or supply contains an error.
Encoder connection is interrupted or defective.
Check wiring of encoder connection.
Encoder defective.
Maloperation due to defective hardware or interference (broken connection, line termination, wiring error, control hardware defective or failed).
