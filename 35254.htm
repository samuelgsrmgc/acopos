35254: SMC: Encoder - Length of physical reference system too large

Info: Additional info
Data type: UDINT

Description:
Error during startup of the encoder.
For the connected encoder the maximum value for "Length of physical reference system for linear encoder (nm)" has been exceeded.

The maximum allowed value is shown in the Additional Information (Info 1).

Offset information for safety logger:
binary data[20..17]: Info1


Reaction:
The status SafePositionValid is set to FALSE.
The status NotErrENC is set to FALSE.
If a safety function is used which needs an encoder, then the module changes into Functional Fail Safe state.

Cause/Remedy:
Check the parameter "Length of physical reference system for linear encoder (nm)" in the SafeDesigner or use an encoder with a lower resolution.
