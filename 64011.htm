64011: Timeout for completion of encoder initialization

Info:
Data type:

Description:
Initial ACOPOS parameters are parameters that are defined for an ACOPOS module in the hardware configuration or for an NC object of that ACOPOS module in an NC mapping table. These ACOPOS parameters are automatically transferred to the ACOPOS in the function for ACOPOS startup.

If the encoder initialization is started after the transfer of an initial ACOPOS parameter, afterwards the corresponding encoder position must not be used before the encoder initialization is completed. Therefore before the transfer of an initial ACOPOS parameter table it is waited, until the encoder initialization is completed. This is indicated by the fact, that the following bit in the general status (parameter STATUS_BITS) is deleted:
- Bit5: Read encoder data active
If the timeout for waiting on this status is reached, then this warning appears.

Reaction:
This warning is indicated.

Cause/Remedy:
Encoder connection is interrupted or defective. See wiring guidelines error number 7030.
