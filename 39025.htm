39025: EnDat encoder: Operating status error sources: M Overflow / Underflow

Info: Slot
Data type: USINT

Description:
Runtime error in an encoder with activated monitor (activation, see error number 7030).
Multiturn Overflow

Reaction:

Cause/Remedy:
Return encoder to the permitted multiturn range and acknowledge error by writing
PARID_ENCOD_CMD with the value 1.
Alternatively the multiturn position can be reset to the current position by writing the value 4 to PARID_ENCOD_CMD.
This causes the encoder to lose its absolute reference, so it needs to be calibrated.
