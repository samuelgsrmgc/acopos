38008: Bleeder: No current flow

Info:
Data type:

Description:
Runtime warning while switching on the controller.

8Vxxxx.00-x:
While the controller is starting up, a test checks to see if the chopper current UDC_CHOP_CURR exceeds the limit value 0.8 * UDC_ACT / R_BLEEDER_EXT when the braking resistor chopper is activated briefly (20ms). The chopper current does not exceed the limit.

8B0Pxxxxxxxx.xxx-x:
While the controller is starting up, a test checks to see if the DC bus voltage UDC_FILTER drops more than 2% when the braking resistor chopper is activated briefly. The DC bus voltage drop during the test procedure was less than 2%.

Reaction:
None

Cause/Remedy:
ACOPOS:
Fuse defective: Replace fuse.
High impedance power line: The forced DC bus voltage drop is less than 2%. The current flow test must be deactivated.
Chopper defective: Replace the ACOPOS module.

External braking resistor:
Wiring error: Check whether all braking resistor lines are properly fastened at the RB+ and RB- connection.
Wiring error: Remove the braking resistor lines from the connections RB+ and RB-. Check the resistance between RB+ and RB- on the braking resistor lines on the ACOPOS side. The resistance must be finite and correspond to the braking resistor data. If the resistance is infinite, then the braking resistor or the line is defective.
Braking resistor defective: Check the resistance between RB+ and RB- right on the braking resistor. If the resistance is infinite, then the braking resistor is defective.

8B0Pxxxxxxxx.xxx-x:
Braking resistor error configuration: The parameter for the resistance value of the braking resistor R_BLEEDER_EXT was set too low.
Chopper current measurement defective: Replace the ACOPOS-module
DC bus voltage measurement defective: The real DC bus voltage value is lower than the measured DC bus voltage value UDC_ACT: Replace the ACOPOS-module

The warning can be deactivated by setting Bit6 (0x40) in MOTOR_TEST_MODE.
