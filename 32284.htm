32284: Ext. coupling data: With this AR/NC version only for TC1 with synchr. output

Info: Number of NC Manager task class
Data type: USINT

Description:
Extended coupling data are used then for a network interface, if "Extended coupling data for ACP10_MC_BROADCAST=Yes" is defined for this network interface in the NC configuration.

With AR versions before A3.09 (respectively before A4.01) or ACP10 versions before V2.310 this is only possible under the following restrictions:
- Cyclic #1 must be defined as NC manager task class
- For this task class, output data must be written in synchronization with the network

If one of these restrictions is not fulfilled when using these versions, this error is indicated.

Reaction:
Processing of the corresponding NC action is aborted.

Cause/Remedy:
Change the AR version to A3.09 or higher (respectively A4.01 or higher) and the ACP10 version to V2.310 or higher.
With these versions extended coupling data can be used under the following conditions:
- See error number 32285
- See error number 32286

If AR versions before A3.09 (respectively before A4.01) or ACP10 versions before V2.310 are to be further used, the configuration must be modified in such a way that the restrictions described above are fulfilled.
Define "Task class for NC Manager task=Cyclic #1" in the NC configuration.
Select for the task class Cyclic #1 in the system configuration that output data is written in synchronization with the network:
- Automation Studio before V3.0.90: "Output without jitter"
- Automation Studio from V3.0.90 on: "I/O output delay=delay to end of cycle"
