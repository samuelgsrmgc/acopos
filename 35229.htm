35229: SMC Functional Fail Safe: Homing - Standstill Speed tolerance violated

Info:
Data type:

Description:
Error during encoder homing with mode "Homing direct" - the standstill speed limit has been violated.

Reaction:
The module changes into Functional Fail Safe state.

Cause/Remedy:
Check the safety application and the configuration in the SafeDESIGNER!
Check the homing parameter "Standstill speed tolerance"!
Check the homing procedure!
