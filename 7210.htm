7210: DC bus: Charging: Voltage unstable

Info:
Data type:

Description:
Runtime error using the 'switch on controller' command.
The main relay is open.
A valid nominal DC bus voltage UDC_NOMINAL could not be detected.
The rate of change of the filtered DC bus voltage UDC_FILTER is greater than 62.5 VDC / s.
The rate of change of the filtered DC bus voltage UDC_FILTER is less than -6.25 VDC / s.
The filtered DC bus voltage UDC_FILTER sporadically sinks below the minimum cut-off voltage 0.9*MAX(UDC_NOMINAL, UDC_DETECTION).
The filtered DC bus voltage UDC_FILTER sporadically exceeds the limit voltage:
8Vxxxx.00-x.......746 VDC (1.1*sqrt(2)*480VAC)
8Vxxxx.50-x.......357 VDC (1.1*sqrt(2)*230VAC)
8BxP.................746 VDC (1.1*sqrt(2)*480VAC)
8BVI.................880 VDC
8CVI.................880 VDC
8DI. .................880 VDC
80V..................88 VDC

Reaction:
See error number 4005.

Cause/Remedy:
See Power transmission

-> Request the status 'controller ready' before the command 'switch on controller'.
Mains disturbance: Large DC bus voltage fluctuation. -> Check the filtered DC bus voltage UDC_FILTER.
