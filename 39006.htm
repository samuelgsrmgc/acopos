39006: EnDat encoder: Alarm bit - Position value contains an error

Info: Slot
Data type: USINT

Description:
Runtime error in an EnDat encoder with activated monitor (activation, see error number 7030). The position monitor internal to the encoder detected an error.

Reaction:
None

Cause/Remedy:
See also error number 39005.
Encoder speed or speed of movement is too high.
Encoder vibrations (mechanical vibrations).
Using battery-buffered encoder: See error number 39010
