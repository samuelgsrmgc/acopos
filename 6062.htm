6062: CTRL Speed controller: Speed error stop limit exceeded

Info: Limit value AXLIM_DV_STOP
Data type: REAL

Description:
Runtime error when controller is active. The absolute value of the speed error SCTRL_SPEED_ERROR exceeds the limit value AXLIM_DV_STOP.

Reaction:
See error number 9001.

Cause/Remedy:
Speed error limit too small (AXLIM_DV_STOP).

Incorrect configuration for set value generation: See error number 4007.
Incorrect controller configuration: See error number 4007.
Current limiting: See error number 4007.
Encoder speed contains an error: See error number 4014.
Motor wiring error: See error number 6044.
Motor wiring error: See error number 6045.
Motor defect: See error number 6045.
ACOPOS defect: See error number 6045.
