35079: SMC Fail Safe: Homing - Position is out of SMP window

Info:
Data type:

Description:
Configuration error of encoder homing. The homing position is outside the SMP window, while homing with mode "homing direct".

Reaction:
The module changes into Fail Safe state.

Cause/Remedy:
Check the parameter "Safe lower position limit for SMP"!
Check the parameter "Safe upper position limit for SMP"!
Check the homing parameter "Homing position or homing offset"!
Carry out a Power Down/Power Up cycle!
