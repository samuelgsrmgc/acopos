32133: The NC module type of the specified NC data module is invalid

Info: Requested type (in HB) and type of NC data module (in LB)
Data type: UINT

Description:
This error is displayed, if an NC data module with a certain NC module type should be accessed, but the specified NC data module has another NC module type.

Reaction:
Processing of the specified NC data module is aborted.

Cause/Remedy:
Specify an NC data module with the correct NC module type.
NC module types:
- 11 NC manager configuration
- 12 NC INIT parameters
- 13 NC error text table
- 16 ACOPOS parameter table
- 20 CNC program
- 21 CNC zero point table
- 22 CNC tool data table
- 23 CNC R-parameter table
- 24 CNC cam profile (table)
- 25 CNC cam profile (polynomial)
- 27 ACOPOS cam profile (polynomial)
- 28 NC mapping table
- 30 ACOPOS network trace
- 31 ACOPOS parameter trace
- 32 MTC data (Motion Trace Configuration)
