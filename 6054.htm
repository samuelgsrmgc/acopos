6054: Power stage: Overcurrent

Info:
Data type:

Description:
Runtime error if at least one of the current sensors (Phase U, V, "W") in the power stage registers overcurrent.

Reaction:
See error number 6019.

Cause/Remedy:
Motor wiring error:
See error number 6019.

Motor defect:
See error number 6019.

ACOPOS servo drive defect:
See error number 6052.
- Problem with current measurement. -> Check the phase current when the power section is switched off.

Set current value jump:
See error number 9300.
