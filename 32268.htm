32268: Trace test data point: The NC object is not enabled for trace commands

Info: Index of test data point
Data type: UINT

Description:
In the following cases an NC object is not enabled for trace commands:
- For the corresponding ACOPOS module the startup was not yet completed
- The correspondig trace channel is already used in trace

Reaction:
Abortion of trace processing.

Cause/Remedy:
Wait until the ACOPOS startup or the trace started before is completed.
