7217: DC bus: Nominal voltage detection: Voltage too high

Info: Maximum DC bus voltage
Data type: REAL

Description:
Runtime error using the 'switch on controller' command.
The main relay is open.
When the controller is turned on, the filtered DC bus voltage UDC_FILTER is checked.
The error is generated, if the filtered DC bus voltage UDC_FILTER is higher than the threshold voltage of the braking resistor UDC_BLEEDER_ON or higher than the following module-dependent limit voltage.
8Vxxxx.00-x:............746VDC = 1.1*sqrt(2)*480VAC
8Vxxxx.50-x:............357VDC = 1.1*sqrt(2)*230VAC
8BxP:......................746VDC = 1.1*sqrt(2)*480VAC
8BVI:......................880VDC
8CVI:......................880VDC
8DI:........................880VDC
80V:.......................88VDC

Reaction:
See error number 7210.

Cause/Remedy:
See Power transmission

Invalid mains: Supply voltage or DC bus voltage is too high. -> Check the DC bus voltage.
