39018: Reference pulse monitoring: Faulty position, resolution, or reference pulse

Info: Slot
Data type: USINT

Description:
Runtime error when incremental reference pulse monitor is active. The distance between two reference pulses is outside the valid range.

Reaction:
None

Cause/Remedy:
Compare configuration of the reference pulse monitor (pulse width, pulse interval, watch window and mode) with encoder data sheet.
See also error number 7047.
