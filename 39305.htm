39305: Digital IO 10: Diagnose bit active (current, temperature)

Info: Slot
Data type: UINT

Description:
Runtime error on digital output 10 of an AC130 plug-in card. The diagnostics but was set due to current overload or thermal overload.

Reaction:
Current overload, or thermal overload causes the output to be shut off.

Cause/Remedy:
See error number 39302.
