32282: Network coupling: Incompatible cycle times (network interfaces, NC task class)

Info:
Data type:

Description:
If coupling master and coupling slave are connected to different network interfaces, it is proofed, wether the following conditions are fulfilled:
- The network cycle time of the slave axis is equal to the cycle time of the NC manager task class
- The network cycle time of the slave axis is greater than or equal to the network cycle time of the master axis
- The ratio of the network cycle times of slave and master axis is integer

If one of the conditions specified above is not fulfilled, this error is indicated.

Reaction:
Processing of the corresponding NC action is aborted.

Cause/Remedy:
Adjust the cycle times listed in the description above, so that all conditions are fulfilled.
