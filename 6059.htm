6059: Enable2: Voltage sag

Info:
Data type:

Description:
Runtime error in the hardware monitor using the 'switch on controller' command or while the controller is active. A voltage sag occurs on the controller enable input (contact ENABLE2 on connector X1).

Reaction:
When 'switch on controller' command: See error number 4005.

When controller is active:
See error number 6023.

Cause/Remedy:
See error number 6023.
