6042: Operating system version is not in the allowed range

Info:
Data type:

Description:
Runtime error when switching on the controller if the minimum required version BSYS_VERS_MIN is lower than the ACOPOS operating system version or the maximum allowed version BSYS_VERS_MAX is higher than the ACOPOS operating system version.

Reaction:
The 'controller ready' state is reset. See error number 4005.

Cause/Remedy:
See error number 6040.
See error number 6041.
