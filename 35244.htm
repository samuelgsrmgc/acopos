35244: SMC: Encoder - Received position not valid

Info: Additional info
Data type: UDINT

Description:
Error at the encoder input. During the reading of the position, an error has occurred.

The additional information (Info1) shows the details:
Bit0 (0x00000001) ... Faulty EnDat transfer timing.
Bit2 (0x00000004) ... Faulty CRC of receive register 1S.
Bit3 (0x00000008) ... Faulty CRC of receive register 4.
Bit4 (0x00000010) ... Faulty CRC of error register.
Bit5 (0x00000020) ... The safety cycle has not been completed.
Bit6 (0x00000040) ... The consecutive number of the data storage register 4 is faulty.
Bit7 (0x00000080) ... The consecutive number of the data storage register 2 is faulty.
Bit8 (0x00000100) ... The consecutive number of the data storage register 3 is faulty.
Bit9 (0x00000200) ... Error bit in error register is set.
Bit10 (0x00000400) ... The forced dynmaic test of the EnDat master is faulty.
Bit11 (0x00000800) ... The forced dynmaic test of the encoder is faulty.
Bit12 (0x00001000) ... Position compare (Pos1/Pos2) is faulty.
Bit13 (0x00002000) ... Error encoder mask is faulty.
Bit14 (0x00004000) ... Encoder parameter read error.

Offset information for safety logger:
binary data[20..17]: Info1

Reaction:
The status SafePositionValid is set to FALSE.
The status NotErrENC is set to FALSE.
If a safety function is used which needs an encoder, then the module changes into Functional Fail Safe state.

Cause/Remedy:
The data transfer is disturbed between the encoder and the module. Check the wiring between the encoder and module!
The encoder is defect. Change the encoder!
The module is defect. Change the module!
