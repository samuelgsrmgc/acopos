35245: SMC: Encoder mismatch detected

Info: Additional info
Data type: UDINT

Description:
Error during startup of the encoder. A change in the encoder configuration data was detected.

The additional information (Info1) shows the details:
0x01 ... The length of the configuration data has changed.
0x02 ... The encoder serial number has changed.
0x03 ... The configuration data version has changed or there is no configuration data.
0x10 ... The EnDat master configuration register 1 has changed.
0x11 ... The EnDat master configuration register 5 has changed.
0x12 ... The EnDat master configuration register 6 has changed.
0x13 ... The EnDat master configuration register 7 has changed.
0x14 ... The EnDat master configuration register 8 has changed.
0x15 ... The EnDat master configuration register 9 has changed.
0x16 ... The EnDat master configuration register 10 has changed.

Offset information for safety logger:
binary data[20..17]: Info1

Reaction:
The module state change not into operational.

Cause/Remedy:
0x02,0x03:
Confirm the encoder exchange on the SafeLogic!
Check the application and/or the encoder.
0x01,0x10-0x16:
Call technical support!
