5024: Cyclic set value mode aborted: Set positions missing

Info:
Data type:

Description:
Runtime error in the operating mode, 'cyclic, external set values'. Linear extrapolation occurs if a new set position (CYC_MASTER_SET_POS) is not received within the master cycle. This error is registered if this failure occurs several times consecutively.

Reaction:
See error number 5036.

Cause/Remedy:
The operating mode 'cyclic, external set values' is activated too early, before set values are sent.
See error number 5110.
