32009: Error sending Write Request (network error ?)

Info: Status of CAN_sendCOB()
Data type: UINT

Description:
For the operation of ACOPOS servo drives via the CAN network, the NC manager first sends a Write Request telegram with the CAN_sendCOB() function from the PLC to the ACOPOS for transferring a parameter value to the ACOPOS. This error number is output if an error occurs with this procedure.

Reaction:
Resets the interface of the corresponding write channel.

Cause/Remedy:
Overload, malfunction or interruption of the network connection or ACOPOS failure.
See also error number 1004.
Status of CAN_sendCOB(): See CAN library error numbers.
