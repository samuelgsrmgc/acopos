35072: SMC Fail Safe: Homing - SMP window is greater than safe absolute encoder range

Info:
Data type:

Description:
Configuration error of encoder homing. The SMP is configured and the SMP position window is greater than the safe absolute encoder counting range.

Reaction:
The module changes into Fail Safe state.

Cause/Remedy:
Check the parameter "Safe lower position limit for SMP"!
Check the parameter "Safe upper position limit for SMP"!
Check the absolute encoder counting range of the connected encoder!
Carry out a Power Down/Power Up cycle!
