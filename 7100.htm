7100: Parameter function not supported. (Module ?)

Info:
Data type:

Description:
Response error while writing a parameter specific to the plug-in card. The addressed plug-in card does not support the parameter.

Reaction:
Parameter is not set.

Cause/Remedy:
The slot addressed by the parameter contains either no plug-in card or the wrong plug-in card.
The plug-in card is not supported.
See also error number 1002.
