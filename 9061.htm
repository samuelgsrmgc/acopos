9061: ACOPOS continuous current: Overload - Limiter active

Info: Load limit
Data type: REAL

Description:
Runtime error if the load of the ACOPOS continuous current load LOAD_CONT_CURR exceeds the limit (see INFO).

Reaction:
See error number 9001

Cause/Remedy:
During the movement stop (see error number 9060), the load of the ACOPOS continuous current increases.
If the movement would be stopped in already in deeper loads, the limit load (see INFO) would not be exceeded.
