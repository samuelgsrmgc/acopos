10500: Induction stop was terminated

Info:
Data type:

Description:
Runtime error causing the induction stop to terminate.


Reaction:
Power element deactivated and holding brake turned on.


Cause/Remedy:
Synchronous motor:
- An induction stop that was triggered by the error 6019: ACOPOS: "Overcurrent" will be terminated
if an overcurrent occurs again.
- An induction stop that was triggered by the error 7200: DC bus voltage: "Overvoltage" will be
terminated if a temperature error or a short-circuit current occurs that is higher than the
maximum motor current or maximum ACOPOS current.
