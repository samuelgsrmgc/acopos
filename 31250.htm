31250: Di/Do Interface: HW Module not OK

Info:
Data type:

Description:
Runtime error in the digital I/O hardware monitoring using the 'switch on controller' command or while the controller is active.
One of the SDC PVs 'iLifeCnt...' of the Di/Do interface is not changed with each SDC cycle. One of the digital I/O hardware modules does not provide new values with each SDC cycle.

Reaction:
See error number 31249.

Cause/Remedy:
Check the SDC Di/Do interface specific PVs.
See the procedure for error number 31249.
