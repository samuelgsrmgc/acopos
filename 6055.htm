6055: Holding brake: Low voltage

Info:
Data type:

Description:
Runtime error while monitoring the holding brake controller. The minimum permissible holding brake output voltage was exceeded for a duration of 500 ms. Module-dependent limit values can be found in the user's manual.

Reaction:
See error number 6029.

Cause/Remedy:
Wiring: Excessive voltage drop between the auxiliary supply module's 24V output and the holding brake output of the inverter module. -> Use multiple expansion modules, cable cross-sections
8B0Cxxxxxxxx.xxx-x: The DC supply to the auxiliary supply module is less than 315VDC
8B0Cxxxxxxxx.xxx-x: The 24V output of the auxiliary supply module is defective -> Replace module
8B0Cxxxxxxxx.xxx-x: Overloaded module -> Use larger module or an additional module
8BVIxxxxxxxx.xxx-x: Faulty holding brake controller or voltage monitor -> Replace module
