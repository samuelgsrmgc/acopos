6056: Holding brake: Low current

Info:
Data type:

Description:
See error number 6029.

Reaction:
See error number 6029.

Cause/Remedy:
Holding brake or external relay does not fulfill the drive specifications (input resistance too large,..). Deactivate monitor respectively set bit 2 in the BRAKE_MODE.
Holding brake connection defective (broken connection, wiring error)
ACOPOS defective (Holding brake control, internal relay)
External relay defective or incorrectly controlled.
Holding brake defective.
