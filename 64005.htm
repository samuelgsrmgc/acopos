64005: Timeout for parameter enable after start of operating system

Info: Status (bits with wrong value for parameter enable)
Data type: UDINT

Description:
During the network initialization (function ACOPOS startup), after start of the operating system for each ACOPOS module is waited, until the following bits in the general status (parameter STATUS_BITS) are deleted:
- Bit4: Network error
- Bit5: Read encoder data active
If the timeout for waiting on this status is reached, then this warning appears.

Reaction:
This warning is indicated.

Cause/Remedy:
Encoder connection is interrupted or defective. See wiring guidelines error number 7030.
Overload, malfunction or interruption of the network connection or ACOPOS failure.
See also error number 1004.
See also error number 1012.
