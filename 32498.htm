32498: PLCopen_CyclicData_TaskClass is lower than zero or higher than maximum value

Info: Maximum value
Data type: USINT

Description:
For SG4 targets in the NC mapping table for each ACOPOS communication channel that task class can be defined, in which the cyclic PLCopen data are to be copied from a program to the POWERLINK network and vice versa. This is possible in the data record of a real or virtual axlis in the column "Additional Data" with the following XML attribute:
- PLCopen_CyclicData_TaskClass=""

If the BR module acp10cfg containes an invalid value for this parameter, then this error is displayed with the maximum task class number as additional information. Moreover, the values defined with PLCopen_CyclicData_TaskClass are entered in the network trace for the corresponding NC objects with ParID TASKCL_CycDat_PLCopen.

Reaction:
The ACOPOS startup function is aborted for this ACOPOS.

Cause/Remedy:
Correct the value defined with PLCopen_CyclicData_TaskClass in the NC mapping table.
