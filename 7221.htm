7221: Mains: Failure

Info: Line frequency
Data type: REAL

Description:
8BVPxxxxxxxx.xxx-x: Runtime error if the observed mains frequency (SCTRL_SPEED_ACT) differs from the mains frequency identified on start-up (see info) by more than MAINS_FREQ_TOL %.

Reaction:
Switching off the power section
The parameter STAT_UDC_POWERFAIL is set to the value 2.
The 'controller ready' state is reset.

Cause/Remedy:
Mains disturbance: At least one phase of the mains line failed.
-> Check voltage using multimeter.

Mains disturbance: The mains voltage is highly distorted or asymmetrical.
-> Check voltage using power analyser.
