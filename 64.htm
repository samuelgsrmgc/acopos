64: Hardware ID in BR module is invalid (data block write)

Info: Hardware ID
Data type: USINT

Description:
Response error in BsLoader mode during the download of an NC system module ('acp10sys.br' or 'acp10bsl.br'). The hardware ID in the BR module header of the NC system module is invalid or incompatible to the ACOPOS hardware.

Reaction:
See error number 25.

Cause/Remedy:
The NC system module cannot be supported by the ACOPOS hardware.
See error number 32018.
