35252: SMC: Position lag error limit exceeded

Info: Lag error [units]
Data type: DINT

Description:
Error during lag error monitoring. Position lag error limit has been violated.

Reaction:
The status SafePositionValid is set to FALSE.
The status NotErrENC is set to FALSE.
If a safety function is used which needs an encoder, then the module changes into Functional Fail Safe state.

Cause/Remedy:
Check the parameter "encoder monitoring position tolerance"!
The connection between motor and encoder slipped through. Check the mechanical connection between the encoder and the motor.
