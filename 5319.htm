5319: Cam profile data not allowed for state 0

Info:
Data type:

Description:
Response error while writing AUT_ST_DATA_INDEX. Only the data index 0 is allowed in the basis state (AUT_ST_INDEX=0). It is not possible to use a cam profile.

Reaction:
None.

Cause/Remedy:
Invalid data index or state index.
In the event that the cam automat being with a cam profile, the respective state can be configured with AUT_START_ST_INDEX (direct start).
