65535: Response error

Info:
Data type:

Description:
If a drive cannot execute the job requested by a read request or a write request, it answers with the parameter ID 65535 and the error record in the parameter data, using the response error telegram. The master can identify the error causing parameter because the response error telegram sent by the drive is a direct response to a request made by the master.

Reaction:
Resets the interface of the corresponding read or write channel.

Cause/Remedy:
The exact cause can be determined from the details entered in the error record.
