32018: Invalid Parameter ID in system module

Info: Parameter-ID
Data type: UINT

Description:
Before transferring a system module to the ACOPOS, the NC manager reads its parameter ID from the module description. One of the following parameter IDs must be entered at the respective memory location:
- BRMOD_BSL: ACOPOS BsLoader
- BRMOD_NCSYS: ACOPOS operating system

Reaction:
Transfer of the system module to the ACOPOS is aborted.

Cause/Remedy:
The software object "acp10bsl" or "acp10sys" handled by the NC manager is not a valid NC system module.

The following system modules are available for the ACP10-SW:
- acp10bsl: ACOPOS BsLoader
- acp10sys: ACOPOS operating system

The names "acp10bsl" and "acp10sys" are reserved for these system modules and therefore cannot be used in the project for other software objects.

See also error number 32013.
